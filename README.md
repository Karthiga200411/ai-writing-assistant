\# ✍️ AI Writing Assistant



A web app that generates professional content instantly using AI.

Built with Python, Streamlit, and Mistral AI.



\## 🌐 Live Demo

👉 \[Try the app here](https://karthiga200411-ai-writing-assistant.streamlit.app)



\## 🌟 Features



\- Choose from 5 content types (blog post, LinkedIn, email, and more)

\- Select tone (professional, casual, persuasive, funny)

\- AI generates ready-to-use content in seconds

\- Download output as .txt file

\- Clean, minimal UI with sidebar guide



\## 🛠️ Tech Stack



\- Python

\- Streamlit

\- Mistral AI API



\## 🚀 How to Run Locally



1\. Clone the repo:

&#x20;  git clone https://github.com/Karthiga200411/ai-writing-assistant.git



2\. Create virtual environment:

&#x20;  python -m venv venv

&#x20;  venv\\Scripts\\activate



3\. Install dependencies:

&#x20;  pip install -r requirements.txt



4\. Add your Mistral API key:

&#x20;  Create .streamlit/secrets.toml and add:

&#x20;  MISTRAL\_API\_KEY = "your-key-here"



5\. Run the app:

&#x20;  streamlit run app.py
